# 📖 O projekcie

Stitch & Petal to fikcyjny sklep internetowy stworzony dla marki oferującej ręcznie wykonane produkty, takie jak biżuteria i maskotki.

Projekt rozpoczął się od przygotowania interfejsu w Figmie, gdzie opracowałam układ strony, typografię oraz identyfikację wizualną. Następnie został wdrożony w WordPressie z wykorzystaniem WooCommerce i Elementora.

Podczas implementacji część elementów została dostosowana do możliwości wykorzystanych narzędzi, przy zachowaniu najważniejszych założeń projektu oraz spójności wizualnej.

Efektem jest responsywny sklep internetowy umożliwiający przeglądanie produktów, dodawanie ich do koszyka oraz przejście przez proces zamówienia.

# ✨ Zakres projektu

- przygotowanie projektu interfejsu w Figmie,
- wdrożenie projektu w WordPressie,
- konfiguracja WooCommerce,
- stworzenie strony głównej oraz podstron,
- przygotowanie sklepu i kart produktów,
- konfiguracja koszyka oraz procesu zamówienia,
- formularz kontaktowy,
- responsywny układ dla komputerów, tabletów i urządzeń mobilnych,

# 📚 Czego nauczyłam się podczas realizacji?

Projekt pozwolił mi przejść przez cały proces tworzenia sklepu internetowego – od projektu interfejsu po wdrożenie gotowej strony.

Podczas pracy rozwinęłam umiejętności związane z:

- projektowaniem interfejsów w Figmie,
- przenoszeniem projektu graficznego do WordPressa,
- konfiguracją WooCommerce,
- budowaniem responsywnych układów w Elementorze,
- organizacją pracy z wykorzystaniem środowisk Dev, Test i Live na platformie Pantheon.
